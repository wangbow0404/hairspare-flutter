# HairSpare 앱 디자인 시스템 가이드

## 📱 프로젝트 개요

**HairSpare**는 미용실 스페어 급구 해결 플랫폼 모바일 앱입니다. 깔끔하고 모던한 디자인을 추구하며, 사용자 경험을 최우선으로 합니다.

---

## 🎨 디자인 철학

- **깔끔하고 모던한 UI**: 불필요한 장식 없이 핵심 기능에 집중
- **투박하지 않은 디자인**: 부드러운 곡선과 적절한 여백으로 세련된 느낌
- **일관성**: 모든 화면에서 동일한 디자인 언어 사용
- **접근성**: 명확한 시각적 계층 구조와 읽기 쉬운 타이포그래피

## 📏 간격 시스템 (Spacing)

**8px 기반 그리드 시스템** (Tailwind CSS 스타일)

- **spacing1**: 4px
- **spacing2**: 8px
- **spacing3**: 12px
- **spacing4**: 16px (기본 단위)
- **spacing5**: 20px
- **spacing6**: 24px
- **spacing8**: 32px
- **spacing10**: 40px
- **spacing12**: 48px

**사용 예시:**
- 카드 내부 패딩: 16px (spacing4)
- 섹션 간 간격: 24px (spacing6)
- 요소 간 작은 간격: 8px (spacing2)

---

## 🔲 둥근 모서리 (Border Radius)

- **radiusSm**: 4px - 작은 배지, 태그
- **radiusMd**: 8px - 작은 카드, 입력 필드
- **radiusLg**: 12px - **기본 카드, 버튼** (가장 많이 사용)
- **radiusXl**: 16px - 큰 카드
- **radius2xl**: 24px - 특별한 카드
- **radiusFull**: 9999px - 원형 (프로필 이미지, 배지)

---

## 🎭 그림자 (Shadows)

- **shadowSm**: `blur: 1px, offset: (0, 1), opacity: 0.05` - 작은 카드
- **shadowMd**: `blur: 4px, offset: (0, 2), opacity: 0.1` - **기본 카드** (가장 많이 사용)
- **shadowLg**: `blur: 10px, offset: (0, 4), opacity: 0.1` - 떠있는 카드
- **shadowXl**: `blur: 15px, offset: (0, 8), opacity: 0.1` - 모달

## 📝 공고 등록 화면 요구사항

### 화면 구조

1. **AppBar**
   - 제목: "공고 등록"
   - 뒤로가기 버튼 (왼쪽)
   - 배경: White

2. **이미지 업로드 섹션** (최상단)
   - 최대 5장
   - 3열 그리드
   - 갤러리/카메라 선택 옵션

3. **입력 필드** (순서대로)
   - 공고 제목 * (필수)
   - 공고 설명 (선택)
   - 지역 * (드롭다운, 필수)
   - 날짜 * (날짜 선택기, 필수)
   - 시작 시간 * (시간 선택기, 필수)
   - 종료 시간 (시간 선택기, 선택)
   - 금액 (원) * (숫자 입력, 천 단위 포맷, 필수)
   - 필요 인원 (명) * (숫자 입력, 필수)
   - 역할 * (드롭다운: 스텝/디자이너, 필수)
   - 급구 여부 (체크박스, 선택)

4. **등록 버튼**
   - 전체 너비
   - Purple-600 배경
   - 하단 고정 또는 스크롤 끝

### 스타일 가이드
- 모든 입력 필드: 위의 "입력 필드" 스타일 적용
- 필수 필드 표시: 라벨에 "*" 추가
- 간격: 필드 간 16px (spacing4)
- 섹션 간격: 24px (spacing6)

## 🎯 특별 주의사항

### 1. 일관성 유지
- 모든 화면에서 동일한 색상, 간격, 타이포그래피 사용
- 버튼과 입력 필드는 항상 동일한 스타일

- 충분한 여백으로 깔끔한 느낌 유지
- 요소 간 최소 간격: 8px

- 명확한 그룹핑으로 정보 구조화

### 4. 인터랙션 피드백
- 버튼 호버/탭 시 시각적 피드백
- 입력 필드 포커스 시 Purple 테두리

- 터치 영역 최소 44x44px
- 스크롤 가능한 영역 명확히 표시
