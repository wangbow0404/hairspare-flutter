## 🎯 테스트 방법

### 1. 브라우저 새로고침
- `Ctrl+R` (Mac: `Cmd+R`) 또는 `F5`
- 또는 브라우저 완전히 닫고 다시 열기

### 2. 회원가입 테스트
- **새 아이디 사용**: `test001`, `test002` 등
- **이메일 없이**: 이메일 필드 비워두고 회원가입
- **이메일 포함**: 이메일 입력하고 회원가입

### 3. 로그인 테스트
- 회원가입한 계정으로 로그인
- 홈 화면으로 이동 확인

### 4. 홈 화면 테스트
- 공고 목록 표시 확인
- 공고 카드 클릭 → 상세 화면 이동
