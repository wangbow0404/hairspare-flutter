# 헤어스페어(HairSpare) 랜딩페이지 제작 의뢰서

## 1. 헤어스페어란?

**헤어스페어**는 미용실(헤어샵)과 프리랜서 헤어 스탭을 연결해주는 **인력 매칭 플랫폼**이에요.

미용실은 예약이 몰리는 날 갑자기 손이 부족하거나, 정직원을 뽑기엔 애매한 자리가 생겨요. 반대로 프리랜서로 일하는 헤어 스탭(디자이너·어시스턴트)은 오늘 당장 일할 수 있는데 자리를 못 찾는 경우가 많고요. 헤어스페어는 이 둘을 앱으로 빠르게 이어줘요.

- **스페어**: 미용실에 단기·급구 형태로 일하러 가는 프리랜서 헤어 스탭 (구직자)
- **샵**: 인력이 필요한 미용실 사업자

## 2. 어떻게 이용하나요?

### 스페어(구직자) 입장
1. 앱에서 원하는 지역·시간대의 공고를 검색
2. "급구" 공고는 오늘 당장 일할 자리도 찾을 수 있음
3. 지원 → 매장에서 승인하면 근무 확정
4. 근무 당일 앱에서 체크인, 완료되면 정산

### 샵(사업자) 입장
1. 공고 등록 (일시·급여·필요 인원 등)
2. 지원자 확인 후 승인/거절
3. 매칭된 스페어와 앱 안에서 채팅
4. 근무 완료 후 정산

## 3. 핵심 기능 (중요도 순)

- **급구 매칭**: 오늘 당장 필요한 인력도 빠르게 연결
- **노쇼 방지 예약금**: 지원 시 소액 예약금을 걸어 양쪽 다 신뢰 있게 근무 진행 (근무 완료 시 전액 환급)
- **실시간 채팅**: 지원부터 근무 조율까지 앱 안에서 소통
- **근무 스케줄·체크인**: 확정된 근무 일정 관리, 출근 체크인
- **모델 매칭**: 헤어 디자이너와 시술 모델(고객)을 연결하는 기능도 있음
- **리뷰·인증**: 매장/스페어 신뢰도를 보여주는 완료 건수·인증 배지
