# HairSpare 스페어 회원가입 Stitch 디자인 브리프

> **사용 방법:** 이 파일 전체를 Stitch 캔버스 프롬프트에 복사·붙여넣기 하세요.  
> **플랫폼:** 모바일 iOS/Android (390×844), 라이트 테마, 한국어 UI  
> **첨부 권장:** 현재 가입 화면, 역할 선택, 모델 매칭 설정(칩 UI), 로그인 화면 스크린샷

---

## 0. Stitch 산출물 요청

1. **유형 선택** — 2카드 + 로그인 링크
2. **스페어·디자이너 가입** — 계정 + 전문가 프로필 + 약관
3. **모델 가입** — 계정 + 모델 프로필 + 사진 업로드 + 약관
4. **모델 사진 업로드 (상세)** — 1~3장 그리드, 대표 사진, 가이드
5. **가입 완료** — 성공 일러스트 + 본인인증 CTA + 홈 가기
6. **상태 변형** — 로딩, 필드 에러, 중복 아이디
7. *(선택)* **모델 홈 목업** — 공고·지원 UI 없는 홈 변형

---

## 1. 제품 맥락

- **HairSpare(헤어스페어)** — 미용 인력·모델 매칭 플랫폼
- 가입 경로 2가지:
  - **스페어·디자이너** — 일자리 탐색·지원·스케줄
  - **모델** — 디자이너 모델 매칭에 노출·채팅 연결
- 가입 시 스텝/디자이너 구분 없음 (통합 폼)

---

## 2. 가입 플로우 (IA)

```
로그인 → 회원가입 → 유형 선택
  ├─ 스페어·디자이너 → 전문가 가입 폼 → 약관 → 완료 → 본인인증 안내 → 홈
  └─ 모델 → 모델 가입 폼 → 사진 업로드 → 약관 → 완료 → 본인인증 안내 → 홈
```

권장 3~4단계. 한 화면 과도한 스크롤 지양.

## 4. 디자인 시스템

- 색: Primary Purple `#9333EA`, `primaryPurpleLight`, stitch 텍스트 토큰
- 참고: StitchFilterChip, StitchStickyBottomBar, HairSpare 심볼
- 입력: 라운드 rect, prefix 아이콘, 에러 helper text
- CTA: 하단 고정 보라 `회원가입` / `다음`
- **피할 것:** 파란 primary 버튼, 과한 그라데이션, 이모지 남발
