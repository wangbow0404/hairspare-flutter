## 1. 진입·목적

### 1.1 진입 경로

```
샵 홈 스크롤
  → 배너 「성공적인 매칭을 위한 꿀팁」 / 「더 많은 지원자를 모집하는 방법」 탭
    → ★ 이번 디자인 대상: 매칭 꿀팁 랜딩 화면
```

**배너 시각 스펙 (이미 구현됨 — 일치 유지)**

| 요소 | 스펙 |
|------|------|
| 배경 | `#EEF2FF`, border Purple 50 |
| 아이콘 | 원형 White 44px, `lightbulb_outline` Purple |
| 제목 | 15px Bold `#191F28` |
| 부제 | 13px `#4E5968` |
| chevron | 우측 `>` |

### 1.2 화면 목적 (비즈니스)

| 구간 | 목적 | 톤 |
|------|------|-----|
| **Part 1** | 공고 품질을 높이는 **실용 팁** 제공 → 신뢰·체류 | 친절한 코치, Purple/Blue 계열 |
| **Part 2** | **급구** 유료 기능 자연스럽게 연결 (업셀) | 이벤트·성과 강조, Red/Orange |

> Part 1은 “미끼”이지만 **내용 자체는 진짜 도움이 되는** 수준으로 작성·디자인할 것.  
> Part 2에서 급구·최상단 노출·지원자 증가 그래프로 **전환**한다.

### 1.3 사용자

- **Shop(미용실 사장·매니저)** — 스페어(구직 미용인) 모집
- “왜 지원이 안 올까?” 고민 중인 사용자
- 급구가 뭔지 모르거나, 일반 공고만 쓰던 사용자

## 4. Part 1 — 콘텐츠·UI 스펙 (무료 꿀팁)

### 4.1 히어로

**H1 (택1 또는 조합)**  
- `더 많은 지원자를 모집하는 방법`  
- `성공적인 매칭을 위한 꿀팁`

**서브카피**  
- `공고만 잘 써도 지원률이 달라져요. 아래 3가지만 챙겨보세요.`

**비주얼**  
- 배너와 톤 맞춘 연보라(`#EEF2FF`) 히어로 또는 White 카드 + 전구 일러스트  
- 과한 일러스트보다 **깔끔한 아이콘 + 짧은 문장**

---

### 4.2 섹션 A — 「공고 내용을 자세히 써주세요」

**핵심 메시지:** 한 줄 공고보다 **구체적 공고**가 지원 2배 이상 (※ 수치는 “많은 사장님들이 체감” 수준 표현, 과장 금지)

**체크리스트 카드 (✓ 아이콘 Purple)**

| 항목 | 설명 카피 |
|------|-----------|
| 하는 일 | 커트 보조 / 샴푸 / 드라이 / 고객 응대 등 구체 업무 |
| 근무 시간 | 시작·종료·휴게 (예: 10:00–19:00, 점심 1시간) |
| 급여 | 시급·일당·인센티브 명시 (예: 시급 1.5만원) |
| 준비물 | 가운·실내화·도구 지참 여부 |
| 복장·매너 | 검은 상의, 담배 매너 등 |

**Before / After 미니 카드 (2열 또는 상하)**

| | 나쁜 예 ❌ | 좋은 예 ✅ |
|---|-----------|-----------|
| 제목 | `스텝 구합니다` | `강남 ○○미용실 오후 스텝 급구 (시급 1.5만)` |
| 본문 | `연락 주세요` | `샴푸·드라이 보조, 14–20시, 주차 가능, 당일 지급` |

---

### 4.3 섹션 B — 「사진 한 장이 지원을 바꿉니다」

**핵심 메시지:** 텍스트만 있는 공고보다 **살롱·작업 환경 사진**이 있으면 지원 문의가 늘어난다.

**좋은 사진 vs 나쁜 사진 — 2열 비교 카드**

| 좋은 사전 ✅ | 나쁜 사진 ❌ |
|-------------|-------------|
| 밝은 매장 내부·작업대 | 흐릿·어두운 실내 |
| 실제 근무 공간 | 스톡·무관한 이미지 |
| 정돈된 도구·의자 | 개인 셀카·얼굴 클로즈업만 |

**촬영 팁 칩 (가로 스크롤 또는 wrap)**

- `창가 자연광`
- `가로보다 세로(9:16)도 OK`
- `로고·간판이 보이면 신뢰↑`
- `인물 없어도 됩니다`

**UI:** 좋은 예 썸네일에 Green 테두리, 나쁜 예에 Red 연한 배경 + ✕ 아이콘

---

### 4.4 섹션 C — 「신뢰를 주는 한 줄」

**3개 미니 카드 (아이콘 + 한 줄)**

| 아이콘 | 제목 | 본문 |
|--------|------|------|
| 📍 | 위치를 정확히 | 지하철·주차·건물 안내까지 |
| 💰 | 급여를 숨기지 않기 | “협의”만 쓰면 이탈↑ |
| 💬 | 답장 속도 | 지원 후 24시간 내 응답 권장 |

**마무리 문장 (Part 2 전환용)**  
- `위 팁만으로도 지원이 늘어날 수 있어요.`  
- `그래도 **오늘·내일 당장** 인력이 필요하다면? ↓`

## 5. Part 2 — 급구 업셀 (전환 구간)

> 기존 **급구 유도 화면**(`shop_job_urgent_upsell_content.dart`)과 **시각 언어 통일**.  
> 단, 이번 화면은 공고 등록 **직후**가 아니라 홈 배너에서 들어오는 **교육+마케팅 랜딩**이다.

### 5.1 Part 2 히어로 (그라데이션)

**배지:** `🚀 급구` (White glass pill, bolt 아이콘)

**헤드라인 (택1)**  
- `오늘 필요한 인력, 오늘 채우세요`  
- `지원이 더 빨리 오는 급구 공고`

**서브카피**  
- `급구 공고는 **최상단 노출**과 **급구 배지**로 스페어에게 먼저 보여요.`

**글래스 스탯 3칸 (히어로 하단, 급구 유도와 동일 패턴)**

| 값 | 라벨 |
|----|------|
| `3×` | 더 많은 지원 |
| `최상단` | 우선 노출 |
| `당일` | 빠른 매칭 |

---

### 5.2 섹션 D — 급구 혜택 3카드

| # | 제목 | 설명 | 아이콘 |
|---|------|------|--------|
| 1 | 최상단 노출 | 스페어 공고 목록 **맨 위** 고정 영역 | `vertical_align_top` |
| 2 | 급구 배지 | 🚀 급구 뱃지로 **시선 집중** | `bolt` / `local_fire_department` |
| 3 | 빠른 지원 | 급한 일정에 맞는 스페어 **우선 유입** | `groups` / `schedule` |

카드: White, shadow-sm, radius 12px, Part 2 구간에서는 Red 50 아이콘 배경 가능

---

### 5.3 섹션 E — 지원자 수 비교 그래프 ★ (필수)

**섹션 제목:** `급구를 쓰면 지원이 이렇게 달라져요`  
**캡션:** `※ 플랫폼 내 유사 공고 평균 기준 예시입니다` (작은 회색 11px)

**차트 타입 (Stitch에서 1안 선택, Flutter는 static 목업 OK)**

**안 A — 그룹 막대 차트 (권장)**

| | 일반 공고 | 급구 공고 |
|---|---------|----------|
| Y축 | 지원자 수 (0~15명 스케일) | |
| 막대 1 | 3~4명 (회색 `#9CA3AF`) | |
| 막대 2 | 9~12명 (Red/Orange 그라데이션) | |
| 라벨 | `사용 전` / `사용 후` 또는 `일반` / `급구` | |

**안 B — Before → After 화살표 + 숫자**

```
일반 공고  평균 3명  ──→  급구 공고  평균 9명
              ↑ 최대 3배 (보조 라벨)
```

**안 C — 7일 라인 차트 (선택)**

- X: 등록 후 일수 (D+0 ~ D+7)
- 두 라인: 일반(회색 점선), 급구(Orange 실선)
- 급구 라인이 초반에 더 가파르게 상승

**차트 카드 UI**

- White 카드, padding 20px, radius 16px
- 범례: ● 일반  ● 급구
- 하단 한 줄: `첫 24시간 지원이 특히 많아요` (Orange 강조)

---

### 5.4 섹션 F — 최상단 노출 (리스트 목업)

**섹션 제목:** `스페어 화면에서는 이렇게 보여요`

**미니 폰 프레임 또는 카드 스택 목업**

```
┌─────────────────────────┐
│ 🚀 급구  [당신의 급구 공고]  │  ← Red border, 맨 위
├─────────────────────────┤
│ 일반 공고 A              │
│ 일반 공고 B              │  ← 연한 회색, 아래로 밀림
└─────────────────────────┘
```

**캡션 불릿**

- 급구 공고는 **목록 최상단**에 노출됩니다
- 🚀 **급구 배지**로 일반 공고와 구분됩니다
- 스페어 **공고별·급구 필터**에서도 우선 노출됩니다
