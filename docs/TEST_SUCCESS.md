# Flutter 앱 테스트 성공!

## 앱 실행 완료

Flutter 앱이 성공적으로 빌드되고 실행되었습니다!

### 접속 방법

브라우저에서 다음 URL로 접속하세요:
**http://localhost:8080**

### 테스트 체크리스트

#### 1. 역할 선택 화면
- [ ] "HairSpare" 타이틀과 "스페어 급구 해결" 서브타이틀 표시
- [ ] 파란색 "스페어로 시작하기" 버튼 표시
- [ ] 보라색 "미용실로 시작하기" 버튼 표시
- [ ] 버튼 호버 시 애니메이션 효과 확인

#### 2. 로그인 화면 (스페어)
- [ ] 스페어 버튼 클릭 → 로그인 화면 이동
- [ ] 아이디 입력 필드 표시
- [ ] 비밀번호 입력 필드 표시 (눈 아이콘으로 표시/숨김)
- [ ] 로그인 버튼 클릭 가능
- [ ] 회원가입 링크 클릭 → 회원가입 화면 이동

#### 3. 회원가입 화면
- [ ] 필수 필드: 아이디, 비밀번호, 비밀번호 확인

- [ ] 회원가입 버튼 동작 확인
- [ ] 로그인 링크로 돌아가기 가능

#### 4. 홈 화면 (로그인 후)
- [ ] 지역 선택 버튼 표시
- [ ] 급구 공고 섹션 표시 (있는 경우)
- [ ] 일반 공고 섹션 표시
- [ ] 공고 카드 클릭 → 상세 화면 이동
- [ ] Pull to refresh 동작

#### 5. 공고 상세 화면
- [ ] 공고 제목, 미용실 이름 표시
- [ ] 날짜/시간 정보 표시
- [ ] 금액 및 예약금 표시

### API 연결 확인

백엔드 서버가 `http://localhost:3000`에서 실행 중이어야 합니다.

확인 방법:
```bash
curl http://localhost:3000/api/auth/me
```

### 서버 중지 방법

Flutter 웹 서버 중지:
```bash
lsof -ti:8080 | xargs kill
```

또는:
```bash
pkill -f "python3 -m http.server 8080"
```
