# Stitch 업무지시서: 자동로그인 후 브랜드 스플래시 화면

## 1. 목표
- **대상**: 앱 실행 시 자동로그인이 성공한 직후, 홈 화면 진입 전에 잠깐(2~3초) 보여주는 브랜드 스플래시 화면 1종
- **레퍼런스**: 오늘의집 인테리어 광고 크리에이티브 (첨부 이미지) — 실사 배경사진 + 후킹 카피 + 장식용 핫스팟 점선 마커 + 하단 브랜드 태그 구조
- **요청**: 레퍼런스의 **레이아웃 구조와 무드(신뢰감 있는 실사 사진 + 질문형 카피)** 는 그대로 가져오되, **색상·브랜드 요소는 전부 헤어스페어 것으로 교체**
- **버전**: 스페어(구직자)용 1종, 샵(사업자)용 1종 — 레이아웃은 동일, **카피 문구만 다름**

---

## 2. 앱 디자인 시스템 (준수 사항)

### 2.1 색상 (HairSpareColors — 2026-07 리디자인 기준, 구버전 퍼플 아님)
- **Brand Primary (berry)**: `#B3355C` — 포인트 마커, 강조 텍스트, CTA
- **Brand Primary Soft**: `#FBEAF0` — 마커 배경, 연한 강조
- **텍스트(사진 위에 얹는 흰 텍스트용) 그림자**: 검정 半투명 그라데이션으로 가독성 확보 (레퍼런스와 동일하게 사진 위 텍스트 하단에 약한 암전 처리 가능)
- **배경**: 실사 사진 (섹션 3 참고), 하단 브랜드 바는 화이트 `#FFFFFF`
- **텍스트**: Primary `#161616`, Secondary `#8D8D91`

### 2.2 간격 · 모서리 · 타이포
- 8px 단위 간격 (8, 12, 16, 24, 32)
- 마커(점) 지름: 큰 것 24px / 작은 것 16px, radius full
- 카피 폰트: 굵게, 24~28px, 흰색, 줄간격 1.3
- 하단 브랜드 바: 14px, medium

---

## 3. 화면 구조 (레퍼런스 순서 그대로 유지, 색상만 교체)

### Step 1: 배경 — 풀블리드 실사 사진
- **레퍼런스**: 리모델링된 집 내부 사진 (복도·주방)
- **헤어스페어 버전**: 실제 헤어샵 시술 현장 또는 스타일리시한 매장 인테리어 사진
- ⚠️ **미확정 — 사용자 확인 필요**: 자체 촬영 소재가 아직 없다면, 임시로 스톡 이미지(라이선스 확인 필요) 또는 실제 파트너 샵 협조 촬영 중 선택 필요. 이 항목은 디자인 착수 전 확정해야 함.

### Step 2: 상단 카피 (질문형 후킹 문구)
- **레퍼런스 톤**: "이 집처럼 리모델링, 내 예산으로 가능할까?" — 부러움(aspiration) + 현실적 의문을 동시에 자극하는 2줄 문구
- **스페어 버전 카피 초안 (택1 또는 디자이너 제안 받기)**:
  - A안: `이 매장처럼 일할 수 있을까?\n오늘 스케줄로 확인해보세요`
  - B안: `이 급여, 내 실력으로 가능할까?\n지금 급구 공고에서 확인해보세요`
- **샵 버전 카피 초안**:
  - A안: `이런 스페어, 우리 매장에도 올까?\n지금 공고 올리고 확인해보세요`
  - B안: `급한 자리, 이 매장처럼 채울 수 있을까?\n지금 공고 등록해보세요`
- 위치: 사진 상단 1/3 지점, 좌측 정렬, 흰색 굵은 글씨

### Step 3: 장식용 핫스팟 마커 + 연결선
- **레퍼런스**: 초록 점 3개(크기 다름) + 점끼리 잇는 대각선
- **헤어스페어 버전**: 점·선 색상을 브랜드 berry(`#B3355C`)로 교체, 나머지 형태(점 크기 차이, 은은한 펄스 링 효과, 대각선 연결)는 동일하게
- 실제 인터랙션 없는 순수 장식 요소 — 탭해도 아무 동작 안 함
- 배치 위치는 사진 구도에 맞춰 디자이너 재량으로 조정 가능

### Step 4: 하단 브랜드 바
- **레퍼런스**: 로고 아이콘 + "오늘의집 인테리어 | 믿을 수 있는 시공"
- **헤어스페어 버전**: 헤어스페어 심볼 + `헤어스페어 | (부제 문구, 예: 매칭 성공률 OO%, 혹은 슬로건)`
- ⚠️ **미확정 — 사용자 확인 필요**: 부제 문구 슬로건 확정 필요

### Step 5: 하단 홈 인디케이터
- 기기 자체 홈 인디케이터 바 표현 (레퍼런스와 동일, 디자인에도 참고용으로만 표시)

---

## 4. 디벨롭 시 유의사항
- **다크 텍스트 가독성**: 사진 위 흰 텍스트가 밝은 배경사진과 겹치면 안 보일 수 있음 → 텍스트 뒤 은은한 그라데이션 암전 처리 필수
- **일관성**: 마커·CTA 색은 berry 한 가지로 통일 (레퍼런스처럼 초록 하나만 쓰는 것과 동일한 절제된 톤)
- **세로 전체화면**: 모바일 세로 방향, 상태바 영역까지 사진이 꽉 차는 풀블리드 레이아웃
- **역할별 2벌**: 스페어용/샵용 배경사진이 달라도 되고 같아도 됨 — 카피만 위 초안 중 확정된 문구로 교체

---

## 5. 산출물 요청
- 스페어용 스플래시 화면 1종 (카피 A/B 중 확정본, 또는 둘 다 시안으로)
- 샵용 스플래시 화면 1종
- 실제 사용 사진 소재가 아직 없다면, 플레이스홀더 사진에 레이아웃만 잡은 시안도 가능
- 색상 HEX·간격(px)은 2번 항목 수치에 맞춰 명시

## 6. 참고 문구 (Stitch에 복사용)
```
자동로그인 성공 직후 잠깐 보여주는 브랜드 스플래시 화면을 디자인해줘.

레퍼런스 무드: 풀블리드 실사 사진 배경 위에 흰색 굵은 2줄 카피 문구,
사진 위에 점 3개(크기 다름)를 대각선으로 잇는 장식용 마커, 하단에
로고+브랜드명+슬로건 바.

1) 배경: 헤어샵 시술 현장 또는 매장 인테리어 실사 사진 (세로 풀블리드)

2) 카피(스페어 버전): "이 매장처럼 일할 수 있을까?" / "오늘 스케줄로
   확인해보세요" — 사진 상단 1/3, 좌측 정렬, 흰색 굵은 글씨, 뒤에
   은은한 암전 그라데이션

3) 마커: 브랜드 컬러 #B3355C 점 3개(크기 다름, 펄스 링 효과) + 대각선
   연결선. 순수 장식, 인터랙션 없음

4) 하단 바: 흰 배경, 헤어스페어 심볼 + "헤어스페어 | (슬로건)" 텍스트

5) 하단 홈 인디케이터 바 표시

같은 레이아웃으로 샵 버전(카피만 "이런 스페어, 우리 매장에도 올까?"로
교체)도 함께 만들어줘.
```

## 7. 미확정 사항 (착수 전 확인 필요)
1. 배경 실사 사진 소재 — 자체 촬영 예정인지, 스톡 이미지 라이선스 구매할지
2. 하단 브랜드 바 슬로건 문구
3. 카피 A/B안 중 최종 확정본 (또는 둘 다 시안 요청 후 택1)
4. 노출 시간(2초/3초 등)과 탭 시 스킵 가능 여부 — 개발 쪽에서 확정 필요한 항목이라 별도로 안내 예정
